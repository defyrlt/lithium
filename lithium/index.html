lithium

lithium is (probably) fast and (hopefully) ergonomic SQL builder.
Attention: stuff is not even close to ready and can break in a flash. Also, everything is built on nightly and wasn't checked on stable/beta.

As for now, lithium provides interface to create a SQL, but it's not responsible for executing it.
lithium does not handle security in any way, it's on your DB driver!

You can find examples in documentation for every struct.
